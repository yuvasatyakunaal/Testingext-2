ðŸš€ Getting Started

To access and utilize the academic resources in this repository, follow these simple steps.

### Prerequisites

Before you begin, ensure you have the following installed:

- **Git:** For cloning the repository.
- **PDF Viewer:** Any standard PDF reader (e.g., Adobe Acrobat Reader, Foxit Reader, or a web browser) to open the `.pdf` files.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/3-1.git
    ```
    (Replace `https://github.com/your-username/3-1.git` with the actual repository URL)
2.  **Navigate into the project directory:**
    ```bash
    cd 3-1
    ```

### Viewing Resources

Once the repository is cloned, you can browse the academic materials:

1.  **Explore the subject folders:**
    ```bash
    ls
    ```
    You will see folders like `AI/`, `CN/`, `CS/`, `EAD/`, `IDS/`, etc.
2.  **Navigate to a specific subject:**
    ```bash
    cd AI/
    ```
3.  **Open the desired PDF document:**
    ```bash
    open "2024-AIDS-AI-Unit1.pdf"
    ```
